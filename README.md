#랜덤함수(book, chapter, verse)
    book : 0이면 월/일 받기 67이면 시/분 받기 1부터 66 사이 n이면 랜덤함수X
    book == 0||67 이라면 랜덤함수 돌려서 book 재설정
    아니라면 그냥 온 정보 그대로
    return text 해서 해당 구절 보내주기
